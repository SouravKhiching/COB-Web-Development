This folder contains all the files related to my Web Development Internship project, "ToDo List".

The files included are:
	i) index.html - Contains the script of the index or Home page.
	ii) style.css - Contains the CSS script which manages the designs used in index.html.
	iii) script.js - Contains the JavaScript code which adds the functions to different elements for the page index.html.
	iv) instructions.html, instructions.css - Contains the script and design of the Instructions page.
	v) about.html, about.css - Contains the script and design of the About page.
	vi) Background.png - Contains the background photo of the website.
	vii) checked.png, unchecked.png - Contains the photos to mark or unmark a task.

Copy all the related files and run the code to see the working of the webpage.
