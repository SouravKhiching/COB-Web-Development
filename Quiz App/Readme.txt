This folder contains all the files related to my Web Development Internship project, "Quiz App".

The files included are:
	i) index.html - Contains the script of the index or Home page.
	ii) style.css - Contains the CSS script which manages the designs used in index.html.
	iii) script.js - Contains the JavaScript code which adds the functions to different elements for the page index.html.
	iv) Background.jpg - The background photo of the webpage.
	v) instructions.html, instructions.css - Contains the script and design of the Instructions page.
	vi) about.html, about.css - Contains the script and design of the About page.
	vii) questions.js - Contains the questions used in the web application.

Copy all the related files and run the code to see the working of the webpage.
