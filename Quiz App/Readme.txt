This folder contains all the files related to my Web Development Internship project, "Quiz App".

The files included are:
	i) index.html - Contains the index or home page of the webpage.
	ii) style.css - Contains the CSS script which manages the designs used in index.html.
	iii) script.js - Contains the JavaScript code which adds the functions to different elements for the page index.html.
	iv) Images folder - Contains the images in jpg format which are used in the webpage.
	v) pages folder -  Contains the additional files for the webpages "Instructions" and "About".

Copy all the related files and run the code to see the working of the codes.
